# YouTube to MP3 Downloader

A Python script to download YouTube videos and playlists as high-quality MP3 files.

## Features

- Download single videos or entire playlists
- High-quality MP3 output (320kbps)
- Custom download location support
- Progress tracking

## Requirements

- yt-dlp

## Installation

```bash
pip install yt-dlp
```

## Usage

```bash
python youtube_downloader.py
```
